1. Метод main() является точкой входа для выполнения программы на Java. JVM автоматически вызывает этот метод при запуске приложения. Он предназначен для инициализации программы и управления её выполнением.

2. String[] args — это массив строк, представляющий аргументы командной строки, передаваемые программе при её запуске. Например, если приложение запускается с указанием параметров, они становятся элементами этого массива.

3. JDK (Java Development Kit) используется для разработки приложений. Содержит инструменты для написания, компиляции и отладки кода.
   JRE (Java Runtime Environment) предоставляет среду для выполнения Java-программ. Она включает JVM и стандартные библиотеки, но не включает компилятор.

4. Размер массива в Java определяется с помощью свойства .length, которое возвращает количество элементов в массиве.

5. System.out.println() добавляет символ переноса строки после вывода текста, переходя на новую строку.
  System.out.print() выводит текст в текущей строке без перехода на новую.

6. Конкатенация строк осуществляется с помощью оператора + или метода concat(). Эти способы объединяют строки, создавая новую строку как результат.

7. void указывает, что метод не возвращает никакого значения после завершения выполнения.

8. static обозначает, что метод или переменная принадлежат классу в целом, а не его конкретному экземпляру. Статические элементы доступны без создания объекта класса.
